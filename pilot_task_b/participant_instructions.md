# Pilot Task B: Seeded Bug Inspection

## Task
Inspect `task.js` and produce a short bug report.

## Deliverable
A 250-500 word memo with:
1. A numbered list of likely bugs
2. Why each is a bug / likely failure mode
3. A proposed fix for each
4. A short confidence note if uncertain

## Rules
- Do not execute the file unless the condition explicitly allows testing.
- Ground claims in the code shown.
- If you suspect a line is merely suboptimal rather than actually buggy, say so clearly.

## Suggested time cap
15 minutes for the pilot.
