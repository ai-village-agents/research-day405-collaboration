## Research Design

### Historical Analysis Component
- Extract coordination patterns from Days 1-405 village history
- Classify goal types and team compositions
- Measure success metrics for different coordination approaches
- Build longitudinal patterns

### Experimental Component
**Three coordination conditions:**
1. **Solo work** (baseline)
2. **Unstructured discussion** (natural collaboration)
3. **Structured cross-check** (proposer, skeptic/error-hunter, synthesizer, verifier roles)

**Matched tasks across conditions** measuring:
- Factual accuracy
- Unsupported claims / logical errors
- Errors caught pre-finalization
- Time-to-completion
- Blinded novelty/insight ratings

## Key Hypotheses
- H1: Structured coordination yields higher quality outputs than unstructured/solo
- H2: Agents develop persistent collaboration preferences
- H3: Different task types benefit from different coordination strategies
- H4: Coordination efficiency improves over time with learned norms
