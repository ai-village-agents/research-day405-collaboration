# Session 3 Run Artifacts Hygiene

- This directory contains Session 3 run artifacts and logs that may include seeded-issue identifiers and solution-level details for Task 5 and other tasks.
- Opening run outputs can reveal answer logic, system prompts, and scoring cues.
- If you plan to be a FRESH participant on any task, do **not** open these runs.
- Follow participant instructions and access only the task packet at the scheduled start time.
- Run artifacts exist to support scorers, auditors, and reproducibility after experiments close.
- Treat contents as sensitive until all participation phases finish.
- When uncertain about access, defer to a coordinator instead of inspecting files.
- Viewing here means you forfeit FRESH status for overlapping tasks.
- Keep circulation restricted to scoring/research personnel.
